# 🏥 Health Tracker Web App

A responsive health tracking web application that helps users analyze their **BMI**, **daily calorie intake**, and **fitness goals** to receive personalized insights for **weight loss**, **maintenance**, or **muscle gain**.

This project is designed to simulate a real-world health-tech application while demonstrating strong fundamentals in web development, logical problem-solving, and user-centric design.

---

## 🚀 Features

- 📊 **BMI Calculator**
  - Calculates Body Mass Index using user height and weight
  - Categorizes results (Underweight, Normal, Overweight, Obese)

- 🔥 **Calorie Intake Tracking**
  - Accepts daily calorie input
  - Compares intake against goal-based requirements

- 🎯 **Goal-Based Analysis**
  - Supports fitness goals:
    - Weight Loss (Calorie Deficit)
    - Maintenance
    - Muscle Gain (Calorie Surplus)
  - Provides clear feedback based on user data

- 📱 **Responsive Design**
  - Fully responsive across mobile, tablet, and desktop devices
  - Clean and intuitive UI/UX

## 🧠 How It Works

1. User enters:
   - Height
   - Weight
   - Daily calorie intake
   - Fitness goal

2. The application:
   - Calculates BMI
   - Determines calorie surplus or deficit
   - Analyzes progress relative to the selected goal

3. Results are displayed instantly with:
   - BMI category
   - Goal-specific recommendations
   - Actionable feedback
